This directory contains logging and data files generated by the application.

# Logs

dh_web.log - Logs all requests, including errors

dh_web_error.log - Errors

access.log - nginx access log

# Generated static files

These are regenerated via cron job every 6 hours.

dh_conferences_works.zip - Public single CSV of all works

dh_conferences_tables.zip - Public multi-CSV set of all tables

private_dh_conferences_tables.zip - Private  multi-CSV set of all tables
